Холодильники.
Подбор по типу

Структура с правильным разделением: колонны теперь во встроенных, морозильная камера осталась в отдельностоящих.

Шаг 1 · тип установки
если встроенный · тип
если отдельностоящий · тип
Что поменялось vs v1
  • «Колонна» перенесена из отдельностоящих во встроенные
  • Колонн стало три: холодильная (полки), морозильная (ящики, ❄), пара (рядом)
  • В каждой пиктограмме теперь визуально показано содержимое — полки для холода, ящики для мороза
  • «Морозильная камера» остаётся в отдельностоящих — это отдельный морозильник (не колонна)
